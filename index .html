Holy Grail Layout:

À la base, le Holy Grail Layout est une page avec un en-tête, un pied de page et trois colonnes.

Le Holy Grail Layout est un classique des problèmes avec CSS: l'objectif est avoir un centre fluide avec des barres latérales de largeur fixe. La colonne centrale doit être chargée en premier dans le HTML et toutes les colonnes doivent être de la même hauteur.

Flexbox règle ce problème facilement !

Flex permet des alignements bien plus simples que le CSS 'classique', y compris le centrage vertical, habituellement un casse tête sans nom.

align-self: flex-start;
justify-content: center;
align-self: flex-end;
Sticky Footer:

Obtenir le footer de la page avec du contenus clairsemé est quelque chose que chaque développeurs a essayer au moins une fois dans sa vie. Et pour la plupart c'est mission accomplie! Pourtant, ces solutions représentent un gros problème si la taille du footer est inconnue.
Flexbox répond parfaitement à ce type de problème. Bien que connus pour la pose horizontal, il fonctionne aussi bien pour la pose vertical. Tout ce que vous avez à faire est d'envelopper les sections verticales dans un conteneur flex et de choisir ceux que vous souhaitez étendre. Ils prendront automatiquement tout l'espace disponible dans leur conteneur.
